# K-Wave Club - Landing (JavaScript)

Proyecto de landing page en Next.js (JavaScript) + TailwindCSS listo para subir a GitHub y desplegar en Vercel.

## How to run locally
```bash
npm install
npx tailwindcss -i ./styles/globals.css -o ./styles/output.css --watch
npm run dev
```
(Or follow the Tailwind setup steps below.)
